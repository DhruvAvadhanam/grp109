Project Scope

Site Suggestion/Idea

  1. Homepage: provides a list of class offerings filterable by category, location, and calendar date.
  2. Class Pages: A class page providing a description, menu, price, and class sessions (location and dates)
  3. Standard Administrative pages: About us, FAQs page, directions, etc.
  4. eCommerce: Allowing customers to select classes (add to their cart), take advantage of promotions/discounts, and pay.
  5. Tentative: Customer Account Management: Allowing customers to log in, view their purchases, change classes if permitted, receive individual or class communications, or view there schedule.
  6. Tentative: Staffing management - where support staff can pick, swap shifts, and manage their schedule.
  • Deena will provide class content to team
  • Project will start basic with home page, menu, and class description pages. Other items may be incorporated as class progresses.
  • Ecommerce/shopping cart are pending.

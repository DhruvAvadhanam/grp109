Project Name:

 Let's Get Cooking

Project Description:

 Create a website that promotes cooking classes and consists of:
  • Homepage: provides a list of class offerings filterable by category, location, and calendar date.
  • Class Pages: A class page providing a description, menu, price, and class sessions (location and dates)
  • eCommerce: Allowing customers to select classes (add to their cart), take advantage of promotions/discounts, and pay.
  • Customer Account Management: Allowing customers to log in, view their purchases, change classes if permitted, receive individual or class communications, or view there schedule.
  • Standard Administrative pages: About us, FAQs page, directions, etc.

Purpose:

 Provide a convenient means for customers to find and register for cooking classes.

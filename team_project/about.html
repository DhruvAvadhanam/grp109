Cooking Classes Project

Project Description

This project is a website designed to promote and sell cooking classes. We used current websites from companies such as (PCC, Whisk, and the Pantry) as inspiration to develop a home page, class description page, main, secondary and top navigational elements, a login page, checkout page, and shopping cart.

Inspiration

The site was inspired by the lead's experience working with a culinary education program at PCC, updating website content, working with the staff and selling classes. There were multiple opportunities to improve layout, design and functionality and we want to attempt the process. PCC, Whisk and the Pantry websites were also used as a guide to the development.

Site Purpose

The site is designed to display the available classes on the homepage, provide more class information on the class description page along with the ability to register, and allow users to purchase the classes from the cart and checkout. One of the goals is to be somewhat simplistic in design and require minimal page transitions and clicking. These sites can be a bit overwhelming in content and the goal is to allow filtering and searching on the homepage to narrow down content for quick consumption. Customers should be able to filter by content, category, date, time and day of the week, etc. There is a log in page, however the profile and account module is not complete.

Intended Audience

The intended audience is folks who enjoy cooking or are looking for a fun activity to engage in. The demographics would include children for kids classes as well as adults in various ages - young adults 20-30, older adults 30-45, and older.

Features

The site features a homepage that displays available cooking classes, a class description page with more details and registration options, a login page, a shopping cart, a checkout page and FAQs page. It allows users to browse, filter, and purchase classes in a simple uncluttered site.

Service Provided

This site makes it easy for all users to find and register for cooking classes by offering filters like date, time, and category. The simple layout and minimal clicks help users quickly find what they are looking for, whether they are beginners or experienced cooks.

Relevance

Cooking classes are a popular way for people to learn new skills, connect with others, and enjoy offline activities. This site supports that by offering a new, user friendly way to explore and sign up for local cooking experiences.
